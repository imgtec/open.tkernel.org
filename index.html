欢迎来到 Open T-Kernel 的主页

Open T-Kernel 是 T-Kernel 的一个分支，功能上基本没有增加或减少，主要对原版的编译方式进行改进，重构或整理代码，重点改进可维护性，可移植性。

关于 Open T-Kernel 目标

  • 使用 KBuild 进行编译。
  • 符合 ANSI C，参考 MISRAC 标准。
  • 支持 ARM ARM926EJ-S，CORTEX-R4，CORTEX-A8 核心。
  • 支持龙芯 LS232。
  • 支持 MIPS mips32 24kc。

获取源码及编译

du@book64:~$ git clone git://git.open.tkernel.org/t-kernel
du@book64:~$ cd t-kernel/tkernel_source/
du@book64:~/t-kernel/tkernel_source$ git checkout vmtronx 
du@book64:~/t-kernel/tkernel_source$ make menuconfig
du@book64:~/t-kernel/tkernel_source$ make
